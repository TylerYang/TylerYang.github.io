Edit Distance

Given two words word1 and word2, find the minimum number of steps required to convert word1 to word2. (each operation is counted as 1 step.)

You have the following 3 operations permitted on a word:

a) Insert a character
b) Delete a character
c) Replace a character

Analysis

For the problem that conform to the characteristics of Dynamic Programming, we need to figure out how to split a big problem into several smaller ones.

The state and the transition are the keys to solve the DP problem.

Assuming we want to convert the word1 "a" to word2 "". Obviously, it will take one step.

From the word1 "a" to word2 "c", it will also take only one step.

How about convert ab to b? Well, from prev conversion we know the minimum step is one. And the last character of word1 equals to word2’s last character. So that means we don’t need to do anything. The answer is still one.
